Divs semanticas do HTML5

Em HTML5, criaram-se as divs semanticas como a "header", "main", "footer", "nav", "section", "aside", "article" para que tenha um sentido na organização do site e nao fique apenas cheio de divs identificadas

Navigation, onde coloca as coisas do menu. Geralmente coloca-se links aqui.

Link Link Link Link
Header, onde coloca o cabeçalho do site
Main, onde coloca-se todo o conteudo principal do site
Section, responsavel pela separação dos conteudos do site
Article, nao entendi mto, me parece que é usado quando vai colocar algo externo no site como uma parte de uma notícia
Aside, usado para escrever algo em torno do article, mas nao se limita a isso
Footer, onde coloca-se o rodapé do site
