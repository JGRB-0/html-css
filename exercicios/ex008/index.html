Principais Formatações

Negrito/Destaque

Para deixar algo em negrito, usa-se a tag "bold(b)"
Exemplo: João Gabriel, mas essa tag é apenas forma, melhor fazer isso no css.

Caso queira negrito semantico, existe a tag "STRONG", que indica pra voce e para o seu codigo que o termo utilizado entre a tag é um termo forte na frase. strong é mais adequado para o html5, ja que essa é uma linguagem semântica

Exemplo: Valorant

Itálico/Ênfase

Nesta frase temos um termo em italico usando a tag não semântica (i)

Nesta outra frase, temos um termo em italico usando a tag semântica (em)
