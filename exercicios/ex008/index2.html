Código Fonte/ Pré-formatação

O comando document.getElementById('teste') é escrito em JavaScript

Pode-se tambem usar a tag(code) para deixar as letras monoespaçadas, caso vá colocar partes de cogidigo no seu site
Exemplo: if num % 2 == 0: print(f'o numero {num} é par!') else: print(f'o numero {num} é impar!)

Para que o codigo fique com os espaçamentos certos, pode-se usar a tag (pre) antes da tag (code)
Exemplo: 

if num % 2 == 0:
    print(f'o numero {num} é par!')
else:
    print(f'o numero {num} é impar!)
