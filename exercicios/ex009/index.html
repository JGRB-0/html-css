Trabalhando com Listas

Listas Ordenadas (ordem faz diferença)

listas ordenadas são chamadas pela tag (ol) e dentro delas, cada item é chamado pela função (li)

Exemplo:
  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Estudar
  5. Almoçar
  6. Ir para a academia
  7. Voltar para casa
  8. Tomar café da tarde

Listas não ordenadas

  • Comprar pão
  • Comprar Leite
  • Comprar compras
  • Comprar presentes

Listas dentro de listas

Pode-se criar listas dentro de listas, fazendo tipo um index de livros

Exemplo:

  1. Linguagens
    1. C++
    2. Python
    3. C#
    4. HTML
    5. CSS
    6. Node.js

  2. Melhores Linguagens
    1. Python
      • Bots
      • Automação de tarefas
    2. JavaScript
      • Desenvolvimento web
    3. C++
      • Microcontroladores

Lista de definição

Listas de definiçaõ sao chamadas pela tag (dl) e seus items sao chamados por (dt) e (dd) que seria respectivamente: termo e descrição.
Exemplo:

HTML
Linguagem de marcação usada para Desenvolvimento web
CSS
Linguagem de marcação usada para desing do HTML
JavaScript
Linguagem de programação usada para desenvolver a interatividade em um site
